# biblioteca-bliante

projeto escolar de banco de dados para controle de livros e empréstimos.

## o que tem no projeto
- **script sql:** criação do banco, tabelas, inserção de dados e consultas.
- **diagrama:** desenho das tabelas e como elas se ligam.
- **dicionário:** explicação do que cada campo faz.

## como usar
1. execute o código do script no mysql workbench.
2. o banco será criado com 10 registros de teste.
3. as 15 consultas ao final do script geram os relatórios solicitados.

feito para a aula de banco de dados.
